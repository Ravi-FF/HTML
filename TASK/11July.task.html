1: website

A website is a collection of many Web pages that share a single domain name. Websites can be created and maintained by business or organization to serve various purposes.

2: Http/Https

HTTP full from of hayper text transfer protocol. Http is a protocol of transfer data form computer to web server. and Https full form of hayper text transfer protocol secure. https provide a secure path to transfer data form computer to web server.

3: client–server

The main difference between client and server is that a client is a machine or a program that requests for services through the web while a server is a machine or a program that provides services to the clients according to the client’s requests.

4: world wide wab

www is the collection of websites and webpage stored in a web server and connected to local computers through the Internet

5: host and hosting

A web hosting service is a type of Internet hosting service that hosts websites for clients, i.e. it offers the facilities required for them to create and maintain a site and makes it accessible on the World Wide Web.

6: domain

a domain name is an unique identity of a website it is used by user to access the website.

7: FTP

The File Transfer Protocol is a standard communication protocol used for the transfer of computer files from a server to a client on a computer network. FTP is built on a client–server model architecture using separate control and data connections between the client and the server.

8: title bar

The title bar is a horizontal bar located at the top of a window in a GUI. It displays the title of the software, name of the current document or file, or other text identifying the contents of that window.

9: extension

The extension is a three- or four-letter abbreviation that signifies the file type. Extensions are important because they tell your computer what icon to use for the file, and what application can open the file.

10: URL

URL in the Internet is the unique address of any file or website that takes the user to the website available on the server.
